## Features

### Add developer with:
- Full Name
- Role (Frontend / Backend / Full-Stack)
- Tech Stack (comma-separated)
- Experience (in years)
- Description
- Join Date
- Profile Image (Cloudinary upload)
- Edit developer details
- Delete developer
- View full developer profile
